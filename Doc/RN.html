Random NumberÂ¶

  • Here are some of the features related to the random number for cuda-RASPA

  • NOTE:

    1. The random numbers (RN) are divided into RN on the CPU vs. on the GPU

    2. See struct RandomNumber in data_struct.h

    3. For the random number on the CPU, they are generated on the fly

      • function get_random_from_zero_to_one() in RN.cpp

    4. For random numbers on the GPU

      • the code first generates X numbers (default 10 million)

      • then the numbers are copied to the GPU

    5. GPU random numbers are used when generating trial positions (first bead + orientations) for CBMC

    6. When X numbers of GPU random numbers are used, the offset is added by X

    7. When the new offset is greater or equal to the size of pregenerated random numbers

      • The code regenerates random numbers on the CPU first, then the new random numbers are copied to the GPU

  • Input Command RandomSeed:

    1. (integer) RandomSeed

    • Usage:

      1. Sets random seed for the random number generator on the CPU
